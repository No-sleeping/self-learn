【代码】+时间复杂度： 给一个字符串，返回出现最多次数的字符及其次数。
```python
def run1(s):
    n = len(s)
    data = {}
    maxEleIndex = 0
    for i in range(n):
        data[s[i]] = list(s).count(s[i])
        if data[s[i]] > data[s[maxEleIndex]]:
            maxEleIndex = i
    return data[s[maxEleIndex]],s[maxEleIndex]

print(run1("asdaaaddsdddddd"))
----
(9, 'd')
```
